Loco's VIM configuration
========================

Intallation
-----------

git clone https://locojay@github.com/locojay/locovim.git ~/.vim

git submodule init
git sumoduel update


#install submodule's using Vundle

start vim and type :BundleInstall

enjoy

Requirements
------------

- ack
- pyflakes
- ropevim
- ctags
- virtualenv / virtualenvwrapper
